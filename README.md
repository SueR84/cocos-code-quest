# 🐶 Coco's Code Quest

A fun, gamified web app that teaches kids how to write HTML from scratch, guided by **Coco the Code Collie**.

Built for three young coders — Daniel, Eabha and Joni — with GAA, hurling, basketball, travel and music themes throughout.

## Play it
👉 **https://suer84.github.io/cocos-code-quest/**

Open the link on an iPad/tablet, then use Safari's **Share → Add to Home Screen** for an app-like icon.

## Features
- 8 levels, one HTML concept each (headings → paragraphs → images → links → lists → structure → colours → free build)
- Live code editor + instant preview
- XP, coder levels, badges, milestone stickers, streaks and a friendly leaderboard
- Three saved profiles (photo + bio), all stored on the device — nothing leaves the iPad
- Coco the mascot cheers them on with personalised encouragement
- Works offline after the first load
